# 🚀 AccuDash - Next.js 16 Financial Dashboard

AccuDash is a modern, high-performance financial management dashboard built with **Next.js 16**, **TypeScript**, and **PostgreSQL**. This project showcases the power of the latest Next.js features, including Server Actions, Streaming, and the new **Proxy (Middleware)** authentication system.

---

## ✨ Key Features

* **Authentication:** Secure login/logout system using **NextAuth.js (Beta)** with the latest `proxy.ts` implementation.
* **Dynamic Dashboard:** Real-time data visualization for revenue, invoices, and customer metrics.
* **Full CRUD Operations:** Create, Read, Update, and Delete invoices with seamless database integration.
* **Server-Side Validation:** Robust form validation using **Zod** and `useActionState`.
* **Search & Pagination:** Optimized data fetching with URL-based search and page navigation.
* **Accessibility (A11y):** Fully accessible forms with ARIA labels and semantic HTML.
* **Modern Styling:** Beautifully designed using **Tailwind CSS** and **Heroicons**.

---

## 🛠️ Tech Stack

* **Framework:** Next.js 16 (Turbopack)
* **Language:** TypeScript
* **Database:** PostgreSQL (via Vercel Postgres)
* **Auth:** NextAuth.js (Beta)
* **Styling:** Tailwind CSS
* **Validation:** Zod
* **Deployment:** Vercel

## 🚀 Getting Started


### 1. Clone the repository
```bash
git clone [https://github.com/med-mostapha/AccuDash.git](https://github.com/med-mostapha/AccuDash.git)
cd AccuDash
```

### 2. Install Dependencies

Before running the project, you need to install the required packages. Using `pnpm` is recommended for faster and more efficient dependency management:

```bash
# Install all dependencies using pnpm
pnpm install
```
